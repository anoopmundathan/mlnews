# mlnews
Fetch latest news from regional language websites

App is live [here](https://mlnewsapp.herokuapp.com/)

## For Development
### Setup
```
$ git clone https://github.com/anoopmundathan/mlnews.git
$ cd mlnews
```
### Install
```
$ npm install
```
### Run
```
$ node server.js
```
Open browser and visit 
```
http://localhost:3000/
```
